## PUREVER란?

PUREVER는 반려동물과의 행복한 이별을 준비하는 웰다잉 서비스입니다. 갑작스러운 이별이 아닌, 천천히 준비할 수 있도록 도와줍니다. 사용자는 30일 동안 매일 새로운 질문을 받고, 반려동물과의 추억을 기록합니다.
